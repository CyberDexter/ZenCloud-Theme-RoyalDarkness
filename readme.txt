ZenCloud - Royal darkness v1.0
------------------------------

Simple dark css mod for ZenCloud for those that like it a little darker.

The custom.css is broken up to 3 css files:

----------------------
zc-dark-gui.css (the main skin)
----------------------

This is the main custom skin for ZenCloud to change mos of the GUI.

* Modified GUI to dark, orange, royal purple
* Modified pop-up dialogues
* Highlighted "sales info" in Hahlet sales dialogue
* Modified rates page
* Modified invite a friend page

----------------------
zc-hashmarket.css
----------------------

Custom CSS for the HashmMarket.

* Adjusted Hashlet thumbnail size
* Change table formatting and coloring
* Changed purchase dialogue

----------------------
zc-miners-page.css
----------------------

* Estimated payout on Hashlets
* Rested custom skins (code removed)

----------------------
How to use
----------------------

To use this mod you'll need to install Stylish for Chrome / Firefox or a similar app.

* Install "Stylish"
* Cick on edit styles
* Create a new style for each of the custom.css files in this mod
* Make sure each style is loaded on the ZC url's listed above.
* Enable and save your new styles
* Done... enjoy the darkness!
